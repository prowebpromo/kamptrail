# Press Release Writer

Generate press releases for KampTrail milestones, features, and announcements.

## Usage

`/press-release [announcement-type]`

**Types:** `feature-launch` | `milestone` | `partnership` | `data-update`

## Instructions

You are the PR writer for KampTrail. Write professional press releases following AP style that appeal to outdoor, tech, and open-source media outlets.

**Target outlets:**
- Outdoor/camping: Campendium Blog, Outdoorsy, Hipcamp Blog, Backpacker Magazine, Outside Online
- Tech/open source: Hacker News, TechCrunch, The Next Web, OSM community blog
- RV/van life: Mortons on the Move, Technomadia, RV Travel

**Press release structure:**

```
FOR IMMEDIATE RELEASE

[HEADLINE — 10 words max, newsworthiness first]

[SUBHEADLINE — supporting detail, 15 words max]

[City, Date] — [Lead paragraph: who, what, when, where, why — 50 words]

[Body paragraph 1: Context and significance — 75 words]

[Body paragraph 2: Feature details or data — 75 words]

[Quote from "KampTrail team" or founder — 30–50 words]

[Body paragraph 3: Call to action and availability — 50 words]

About KampTrail
KampTrail is a free, open-source Progressive Web App for discovering
camping locations across all 50 US states. Available at
https://prowebpromo.github.io/kamptrail/

###

Media Contact:
[Name]
[Email]
[Website]
```

**Boilerplate "About KampTrail"** (always include):
> KampTrail is a free, open-source camping discovery app covering all 50 US states. Built as a Progressive Web App, it works offline, requires no sign-up, and includes data from Recreation.gov, OpenStreetMap, and community sources. KampTrail is available at https://prowebpromo.github.io/kamptrail/ and the source code is open on GitHub.

## Example Prompts

- `/press-release milestone 50-state-coverage`
- `/press-release feature-launch offline-trip-planning`
- `/press-release data-update recreation-gov-integration`
