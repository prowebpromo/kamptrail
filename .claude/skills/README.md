# KampTrail Marketing Skills

A collection of Claude Code skills for generating marketing content for KampTrail — the free, open-source US camping discovery PWA.

## Available Skills

| Skill | Command | Description |
|---|---|---|
| Social Media Post | `/social-media-post` | Instagram, Twitter, Facebook, Reddit posts |
| SEO Blog Post | `/seo-blog-post` | Long-form SEO content for organic traffic |
| Email Newsletter | `/email-newsletter` | User newsletters (welcome, updates, digests) |
| Ad Copy | `/ad-copy` | Paid campaign copy for Google, Facebook, Reddit |
| Press Release | `/press-release` | PR for milestones, features, and announcements |
| Partnership Outreach | `/partnership-outreach` | Emails for brands, creators, and data partners |
| App Store Description | `/app-store-description` | Store listings for Google Play, PWA directories |

## Quick Start

```bash
# Generate an Instagram post about the new overlays feature
/social-media-post instagram new-feature overlays

# Write an SEO blog post targeting "free camping USA"
/seo-blog-post dispersed-camping-guide "free camping USA"

# Draft a welcome email for new users
/email-newsletter welcome new-user-onboarding

# Create Google Search ad copy
/ad-copy google-search app-installs

# Write a press release for 50-state milestone
/press-release milestone 50-state-coverage
```

## Brand Overview

**KampTrail** is a free, open-source Progressive Web App for discovering camping locations across all 50 US states.

- **URL:** https://prowebpromo.github.io/kamptrail/
- **GitHub:** https://github.com/prowebpromo/kamptrail
- **License:** MIT
- **Stack:** Vanilla JS, Leaflet.js, OpenStreetMap, Recreation.gov data

**Key differentiators:**
- Completely free — no subscription, no ads, no sign-up
- Works offline (Service Worker PWA)
- All 50 US states covered
- Open source and community-driven
- Data from Recreation.gov, OSM, OpenCampingMap, and more

**Primary audience:** Dispersed campers, boondockers, van lifers, overlanders, RV travelers, budget campers, and outdoor enthusiasts across the US.
